## Installation

### Front-End

  1. `cd app`
  2. `cd frontend`
  3. `bash`
  4. `nvm use 16`
  5. `zsh`
  6. `npm install`
  7. `npm start`
  
### Back-End

  1. `cd app`
  2. `cd backend`
  3. `bash`
  4. `nvm use 16`
  5. `zsh`
  6. `npm install`
  7.   Configurar seu MongoDb na pasta `backend/Models/Connection`
  9. `npm run dev`
